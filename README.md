### 架構

#### **概覽**：

- `main_basic.py` 基本語法
- `main_senior.py` 進階語法

#### **main_basic.py**：
- 將在本檔案中，以白話文的方式，解釋程式碼，帶你以 人類視角 的方式，轉換到 電腦的角度，讀懂程式碼。
#### **main_senior.py**：
- 本檔案為 進階語法 的實作檔案，不提供任何解釋
- 若想要了解本檔案的語法，可以參考 [Python 官方文件](https://docs.python.org/zh-tw/3/)
- 若想要了解本檔案的內容，可以參考 [Selenium 官方文件](https://selenium-python.readthedocs.io/)
